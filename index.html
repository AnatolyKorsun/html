    разрешает перенос елементов из основной оси на новую строку(обратный порядок строк)

    елементы выравниваются от начала главной оси(default)

    елементы выравниваются по центру главной оси

  • justify-content: space-between

    елементы выравниваются по главной оси, растояния между парами одинаково, последние по краях

    елементы выравниваются по главной оси, растояния между парами одинаково, последние по краях половина растояния

    елементы выравниваются по главной оси, растояния между парами и по концах одинаково

  • position: relative

    Относительное, предидущего положения, в потоке(от верхнего левого угла). Применимо к елементу. Доступно top:, left:, right:, bottom:.Совмещение противоположных сторон недоступно.После сдвига место останутся от елемента

  • position: absolute

    Абсолютное, поднимает над потоком,соседние заполняют место. Доступно top:, left:, right:, bottom:(от границ елемента). Применимо к елементу, отсчет от первого предка со свойством отличным position:static.

  • position: absolute

    При использовании противоположных значений top + bottom и left + right растянет елемент.

  • position: fixed

    Фиксация относительно окна браузера.Доступно top:, left:, right:, bottom:.При использовании противоположных значений top + bottom и left + right растянет елемент относительно окна браузера.

  • position: sticky

    Липкая фиксация относительно окна браузера в блоке родителя. Блок покидает екран с елементом.Применимо к елементу. Доступно top:, left:, right:, bottom:.Совмещение противоположных сторон недоступно.

  • position: static

    Позиционирование по умолчанию. Для отмены других.

  • z-index:1

    Для елементов с position перекравающих друг друга. Высше номер - высше елемент. Доступно отрицательное значение.

  • transform: translateX(100%)

    сдвигает елемент по оси Х относительно себя

  • transform: translateY(100%)

    сдвигает елемент по оси Y относительно себя

  • transform: translateX(100%) translateY(100%)

    сдвигает елемент по оси Xи Y относительно себя

  • floattransform: translateX(100%) translateY(100%)

    сдвигает елемент по оси Xи Y относительно себя
